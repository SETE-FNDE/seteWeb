Estimativa de Custo Operacional das Rotas

Rotas cadastradas
Esta ferramenta possibilita calcular e detalhar a estimativa de custo operacional de uma rota cadastrada no SETE.
Para calcular o custo, a ferramenta utiliza os dados dos outros módulos do SETE.
  • Para calcular o custo de uma rota selecione a mesma e clique no ícone
  • Para que seja possível calcular o custo da rota é necessário fornecer o número de alunos, sua quilometragem, salário dos envolvidos (motoristas, monitores), custos dos veículos (IPVA, seguro, etc).
  • A ferramenta também utiliza os parâmetros gerais, como o preço do combustível e vida útil dos veículos.
  • Para editar tais parâmetros utilize a ferramenta parâmetros do módulo custo.
